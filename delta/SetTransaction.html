SetTransaction

SetTransaction is an action that denotes the committed version for an application.

SetTransaction is created when DeltaSink is requested to add a streaming micro-batch (for STREAMING UPDATE operation idempotence at query restart).

Creating SetTransaction Instance

SetTransaction takes the following to be created:

  • Application ID (e.g. streaming query ID)

  • Version (e.g micro-batch ID)

  • Last Updated (optional) (e.g. milliseconds since the epoch)

wrap Method

wrap: SingleAction
wrap is part of the Action contract to wrap the action into a SingleAction for serialization.

wrap simply creates a new SingleAction with the txn field set to this SetTransaction.
